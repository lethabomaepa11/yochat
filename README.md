# YoChat!

## Overview
This is a simple realtime chat application designed for simplicity and speed, allowing users to chat in realtime.
Live here: [https://lethabomaepa11.github.io/yochat](https://lethabomaepa11.github.io/yochat)

## Functional Requirements
1. User Authentication
    - Users must be able to create an account, login and logout
    - Users must have a unique username
2. Real Time Features
    - Send/Receive Messages in real-time
    - Support Group and Private Chat
    - Display Online Users
3. Data Persistence
    - Persist chat messages using localStorage to retain messages after page refresh.
    - Store user data in JSON for long-term storage.
4. User Experience
    - Display timestamps for each message.
    - Provide a user profile view.
    - Validate non-existing users upon login.
     

## Setup instructions.
To set up you will need:
- VS Code
- Live Server Preview Extension
- Then you are good to go, now follow the following steps:
1. Clone this repository
   ```bash
   git clone https://github.com/lethabomaepa11/yochat.git
   ```
2. Open the repo folder on your machine
3. Open `index.html` in the folder
4. If you have the live server preview by windows, you will be able to run the preview inside VS Code.
   
## Usage guidelines.
- Access the [link to the chat app](https://lethabomaepa11.github.io/yochat)
- Click login or signup on the navbar
- Once authenticated
- You will be redirected to the chat page.
- If you have no one to chat to yet, click people, then select a person to chat with
- You can click on groups, then create a new group, you will need to select at least 2 people to create a group.
